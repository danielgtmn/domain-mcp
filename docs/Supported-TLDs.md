## How support works

| Layer | Coverage | Notes |
|-------|----------|--------|
| **RDAP (this table)** | All rows below | Preferred path — structured JSON |
| **whoisit overrides** | Marked `override` / `override*` | Community endpoints (e.g. `.de`) |
| **WHOIS fallback** | Most other TLDs | Via IANA `whois:` referral — **not listed** (thousands of names, text formats vary) |

- `override` = TLD only present via whoisit override (not in IANA bootstrap).
- `override*` = IANA had an endpoint; whoisit override **replaces** the URL.
- A TLD **missing** here may still work via WHOIS (`method: whois` in results).

## Availability disclaimer

“Available” means no registration record was found. It is **not** a purchase guarantee. Premium, reserved, or policy-blocked names may still be unregistrable.
